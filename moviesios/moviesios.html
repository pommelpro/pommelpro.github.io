Movies (iOS)

Main Page

  • Clicking "Sort by Date" will reorder the movie list to show the most recently added movie first
  • Clicking on a movie title will bring you to the "Update Movie" page
  • Clicking "+" in the top right corner will bring you to the "Add Movie" page
  • Clicking in the "Search Here" box will allow you to filter the list of movies based on title, sort by, and tags

New Movie Page

  • You can enter the title of the movie, the name you want to use to sort, and any tags to search the movie by
  • Clicking "Cancel" will close the page and bring you back to the main page without saving anything
  • Clicking "Add It" will store the movie in the online database and return to the main page

Update Movie Page

  • This page is populated with detailed information about the row that was clicked on the main page
  • Clicking the text in the top left will return the you to the main page without saving anything
  • You can change the text in the boxes
  • Clicking "Update" will overwrite the previously stored information about the movie with whatever is in the three boxes
  • Clicking "Delete Movie" will remove the movie information from the online database
